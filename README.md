# Bike Sales Analysis Dashboard (Excel)

## Project Overview

This project analyzes customer demographic data to identify patterns and factors influencing bicycle purchases. The goal is to uncover key customer segments, behavioral trends, and business insights that can support targeted marketing strategies and sales optimization.

The analysis was performed entirely in Microsoft Excel using data cleaning, transformation, pivot tables, and interactive dashboards.

---

## Objectives

* Identify customer segments most likely to purchase a bike
* Analyze the relationship between income, commute distance, age, and purchasing behavior
* Provide actionable insights for marketing and sales decision-making
* Build an interactive dashboard for business users

---

## Dataset Information

The dataset contains demographic and behavioral information including:

* Marital Status
* Gender
* Income
* Number of Children
* Education
* Occupation
* Home Ownership
* Number of Cars
* Commute Distance
* Region
* Age
* Purchased Bike (Yes/No)

---

## Data Preparation

Steps performed:

* Data cleaning and formatting
* Creation of derived columns:

  * Age Group
  * Income Band
* Removal of irrelevant fields (e.g., ID for analysis)
* Data validation and consistency checks

---

## Key Performance Indicators (KPIs)

* Overall Bike Purchase Rate (%)
* Purchase Rate by Age Group
* Purchase Rate by Income Band
* Purchase Rate by Region
* Purchase Rate by Commute Distance
* Average Income: Buyers vs Non-Buyers
* Purchase Distribution by Occupation

---

## Dashboard Features

* Interactive filters using slicers
* Demographic segmentation analysis
* Comparative charts (buyers vs non-buyers)
* KPI summary cards
* Region-wise analysis

---

## Key Insights

* Customers with shorter commute distances show higher purchase likelihood.
* Middle-aged customers with moderate to high income represent the strongest buyer segment.
* Professionals and skilled workers demonstrate higher purchasing behavior.
* Home ownership shows positive correlation with bike purchases.
